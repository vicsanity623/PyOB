2. Architecture Overview

PyOB is structured as a modular Python package located in src/pyob/.

CoreUtilsMixin (core_utils.py)
├── Provides: Smart Sleep, Headless Approval, XML Engine, LLM Streaming
│
PromptsAndMemoryMixin (prompts_and_memory.py)
├── Provides: Rule-based Templates (Rule 7), CRUD Memory
│
ValidationMixin + FeatureOperationsMixin (reviewer_mixins.py)
├── Provides: Ruff/Mypy validation, Runtime Auto-Heal, XML Implementation

9. LLM Backend & Smart Sleep Backoff

PyOB rotates through a pool of up to 10 Gemini API keys. Keys hitting 429 Rate Limit are quarantined for 20 mins.

Formula: sleep_duration = min(key_cooldowns) - current_time

15. Troubleshooting

ModuleNotFoundError: No module named 'src'

AI incorrectly added src. prefix. Fix by ensuring Rule 7 is enforced in the prompt template.

termios.error: Inappropriate ioctl for device

PyOB tried to call input() in CI. Ensure Headless Auto-Approval is active.
